# ABOAT

An automatic Big-O Analysis Tool(incompletion)

## How to use
```
cd <where you want ABOAT to live>
git clone https://github.com/ncu-psl/ABOAT.git
cd ABOAT
gradle run
cat Big-O_Report.txt
```
